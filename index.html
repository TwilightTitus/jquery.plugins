Welcome to JQuery - Titus Lib .

This javacript/JQuery library provides some useful features.

  • you get a template language -> Javascript Template Language - JSTL. This similar to Java Standard Tag Library
  • you get a logging system similar to log4j in java.

And all this for the client side of a web application.

For more information and a sample, make a download and read the doc.
